# 仿 jira 项目

使用 react hooks + Typescript + ant design 构建.

## 安装依赖

```
yarn install
```

## 启动项目

```
yarn start
```

## 测试项目

```
yarn test
```

使用`jest`和`@testing-library/react-hooks`进行代码测试

## 打包项目

```
yarn build
```

## 部署项目

```
yarn deploy
```

使用`gh-pages`提交并部署到 github page

使用`spa-gh-pages`解决单页应用跳转 404 问题
